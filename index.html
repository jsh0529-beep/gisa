Anthropic API 키 발급받아 등록하는 법

  1. console.anthropic.com에 접속해 계정을 만들고 로그인합니다. (구글 계정으로 가입 가능)
  2. 좌측 메뉴 Billing에서 결제 카드를 등록하고 크레딧을 충전합니다. 최소 $5부터 가능하며, 이 앱 기준 $5면 기사 수십 건을 쓸 수 있습니다.
  3. 좌측 메뉴 API Keys → Create Key를 누르고 아무 이름이나 붙여 키를 만듭니다.
  4. 생성된 sk-ant- 로 시작하는 키를 복사합니다. 키는 이 화면에서 단 한 번만 보여주므로 창을 닫기 전에 꼭 복사하세요. (놓쳤다면 새 키를 만들면 됩니다)
  5. 이 페이지 상단 검은 바에 키를 붙여넣고 저장을 누릅니다. 초록 불이 켜지면 준비 완료입니다.

키는 지금 쓰는 이 브라우저에만 저장되며 외부로 전송되지 않습니다. 단, 키를 아는 사람은 누구나 내 크레딧으로 API를 쓸 수 있으니 키를 남에게 공유하거나 GitHub 코드에 직접 넣지 마세요. 유출이 의심되면 console의 API Keys에서 해당 키를 삭제하면 즉시 차단됩니다.

비용은 사용량만큼 차감됩니다. 대략 기사 1건 작성에 10~30센트, 이슈 리스트는 랭킹·RSS 직접 수집이라 대부분 무료입니다. 사용량은 console의 Usage 메뉴에서 확인할 수 있습니다.
